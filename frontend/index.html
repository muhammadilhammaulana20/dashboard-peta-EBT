REKOMENDASI STRATEGIS

Rekomendasi Kebijakan

Berdasarkan hasil analisis data dan model skoring PETA-EBT untuk mempercepat program dedieselisasi Desa 3T.

Prioritaskan BPP Tinggi

Desa dengan BPP diesel di atas Rp3.800/kWh (Papua, Maluku, NTT) harus jadi prioritas utama konversi ke EBT. Penghematan operasional tahunan mencapai puluhan miliar rupiah.

Optimalkan Potensi Lokal

Prioritaskan PLTS + Battery untuk mayoritas desa 3T. Untuk desa dengan aliran sungai stabil, mikrohidro lebih ekonomis jangka panjang. Sesuaikan dengan potensi dominan tiap desa.

Integrasi ke ESDM One Map

Output PETA-EBT sebaiknya terintegrasi sebagai modul pertimbangan dalam ESDM One Map yang sudah ada, bukan platform baru terpisah, untuk kemudahan adopsi pemerintah.
